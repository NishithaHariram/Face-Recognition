The objective of this project is to identify and recognize people from facial images using deep learning techniques. The images are obtained from the Labelled Faces in the Wild (LFW) dataset, which contains photographs of various individuals captured under different real-world conditions. The project uses advanced facial recognition models to detect and classify faces accurately.

To begin, the LFW dataset was downloaded locally and uploaded to the Google Colab runtime environment. All the required Python libraries were imported, and access to Google Drive was configured. This allows trained models and generated data to be saved and reused later, eliminating the need to retrain the model from scratch every time the project is executed.

During preprocessing, images are converted from the BGR color format used by OpenCV to the RGB format required by the deep learning models. Face detection is then performed using MTCNN (Multi-task Cascaded Convolutional Neural Network), which locates facial regions within each image. To ensure sufficient training data for each individual, only people with at least 15 images were considered. After applying this criterion, the dataset contained 96 individuals for model training and evaluation.

The facial recognition system follows a deep learning-based approach. MTCNN is used for face detection, while a pre-trained EfficientNet Lite0 model from TensorFlow Hub extracts facial embeddings that capture unique facial features. These embeddings are used to train a Support Vector Machine (SVM) classifier with a linear kernel. During prediction, a new image undergoes the same detection and feature extraction process, after which the trained SVM classifier identifies the most likely person and provides a confidence score. The final results, including bounding boxes and prediction labels, are displayed using OpenCV and Matplotlib.

One of the major challenges encountered during development was accessing the dataset directly from online sources, which resulted in repeated connection and download issues. As a result, the dataset was uploaded manually to Colab for a more reliable workflow. To run the project, the necessary libraries such as TensorFlow, TensorFlow Hub, OpenCV, MTCNN, Scikit-learn, and LZ4 must be installed. The dataset is then extracted, processed to generate embeddings, and used to train the SVM classifier. The trained model is saved to Google Drive and can later be loaded to perform predictions on new images, providing a complete pipeline for face detection, feature extraction, classification, and visualization.

The trained model achieved an accuracy of 96.42% on the training and evaluation dataset. However, during testing with previously unseen images, the recognition accuracy dropped significantly, often resulting in the faces being classified as "Unknown." To address this issue, the confidence threshold was adjusted so that a face is labeled as "Unknown" only when the prediction confidence is below 0.05. If the confidence exceeds this threshold, the model displays the predicted identity.

Since the problem involves multi-class classification with a large number of individuals, the confidence scores are naturally distributed across many classes. As a result, the highest confidence value for a prediction is often around 20% rather than close to 100%. The SVM classifier calculates confidence scores for all possible classes, ranks them internally, and selects the class with the highest score as the final prediction. Therefore, even relatively low confidence values can correspond to correct predictions when many classes are involved.

Once, i had ran 100 images using the model the accuracy obtained was 97.92%, thus stating it is an excellent model.
